Select

Useful for allowing users to select only one option from a set.

<Select>
  <Option value="option1" label="Option one" />
  <Option value="option2" label="Option two" />
  <Option value="option3" label="Option three" />
</Select>

Props

PropTypeDescription
childrenOption | OptionGroupThe options or option group to render
onValueChangeFunction (value: string)A function to be called when the value changes
size0 | 1The size to apply
variantnormal | ghostThe variant to apply

System props

Prop
margin
width

Option

Props

PropTypeDescription
valueStringThe value to apply
labelStringThe label to display

OptionGroup

Props

PropTypeDescription
childrenOptionThe options to render
labelStringThe group label to apply
disabledBooleanWhether it is disabled or not

Examples

Variant

There are 2 variants options to choose from.

<Select variant="normal">
  <Option value="option1" label="Option one" />
  <Option value="option2" label="Option two" />
  <Option value="option3" label="Option three" />
</Select>
<Select variant="ghost" ml={6}>
  <Option value="option1" label="Option one" />
  <Option value="option2" label="Option two" />
  <Option value="option3" label="Option three" />
</Select>

Sizes

There are 2 sizes to choose from.

<Select size={0}>
  <Option value="option1" label="Option one" />
  <Option value="option2" label="Option two" />
  <Option value="option3" label="Option three" />
</Select>
<Select size={1} ml={6}>
  <Option value="option1" label="Option one" />
  <Option value="option2" label="Option two" />
  <Option value="option3" label="Option three" />
</Select>

With groups

Group options using the OptionGroup Component

<Select size={0}>
  <Option value="option1" label="Option one" />
  <Option value="option2" label="Option two" />
  <Option value="option3" label="Option three" />
  <OptionGroup label="Option group">
    <Option value="option-group1" label="Option group 1" />
    <Option value="option-group2" label="Option group 2" />
    <Option value="option-group3" label="Option group 3" />
  </OptionGroup>
</Select>
