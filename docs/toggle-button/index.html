ToggleButton

Useful for allowing users to select one or more options from a set.

<Box width="135px">
  <ToggleButtonGroup value="left" onChange={alert}>
    <ToggleButton value="left">Left</ToggleButton>
    <ToggleButton value="center">Center</ToggleButton>
    <ToggleButton value="right">Right</ToggleButton>
  </ToggleButtonGroup>
</Box>
ReferenceError: alert is not defined

ToggleButtonGroup Props

PropTypeDescription
childrenReactNodeThe content to render
valueanyThe current selected value
onChangefunctionCallback triggered with the selected ToggleButton value as parameter

ToggleButton Props

PropTypeDescription
childrenReactNodeThe content to render
