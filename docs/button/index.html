Button

Useful to communicate actions to your users.

<Button>New project</Button>

Props

PropTypeDescription
childrenReactNodeThe content to render
variantgray | blue | green | redThe variant to apply
size0 | 1The size to apply
isWaitingBooleanWhether it is waiting or not

System props

margin

Examples

Colors

There are 4 colors to choose from.

<Button variant="gray">
  Cancel
</Button>
<Button variant="blue" ml={6}>
  Save
</Button>
<Button variant="green" ml={6}>
  Copy
</Button>
<Button variant="red" ml={6}>
  Delete
</Button>

Sizes

There are 2 sizes to choose from.

<Button size={0}>
  Default
</Button>
<Button size={1} ml={6}>
  Large
</Button>

Waiting state

When the user is waiting for the server to respond.

<Button isWaiting>Waiting</Button>
