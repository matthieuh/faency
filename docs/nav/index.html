NavGroup Props

PropTypeDescription
childrenReactNodeThe content to render
variantleft | rightThe position of children in the navigaion bar

NavItem Props

PropTypeDescription
childrenReactNodeThe content to render
variantactive | normalThe navigation state of this item
asa | buttonThe HTML element to use to render the navigation item
