## Examples

### NavGroup Variant

There are 2 variants to choose from.

```js live
<Nav>
  <NavGroup variant="right">
    <NavItem>
      <Text size={2} textColor="grays.5" fontWeight={600}>
        Button on the right
      </Text>
    </NavItem>
  </NavGroup>
</Nav>
```

```js live
<Nav>
  <NavGroup variant="left">
    <NavItem>
      <Text size={2} textColor="grays.5" fontWeight={600}>
        Button on the left (default)
      </Text>
    </NavItem>
  </NavGroup>
</Nav>
```

```js live
<Nav>
  <NavGroup variant="right">
    <NavItem>Button on the right</NavItem>
  </NavGroup>
  <NavGroup variant="left">
    <NavItem>Button on the left</NavItem>
  </NavGroup>
</Nav>
```

### NavItem As

There are 2 variants to choose from.

```js live
<Nav>
  <NavGroup variant="left">
    <NavItem as="button" onClick={() => alert('clicked!')}>
      As a button
    </NavItem>
  </NavGroup>
</Nav>
```

```js live
<Nav>
  <NavGroup variant="left">
    <NavItem as="a" href="https://containo.us">
      As a link
    </NavItem>
  </NavGroup>
</Nav>
```
